PURPOSE

Research project exploring AI agent autonomy. Goals:

  1. Understand how personality emerges in agents (memory, prompts, context)
  2. Research techniques for agent independence (fine-tuning, RAG, local models)
  3. Eventually train a custom model that embodies the agent's personality
  4. Document the journey for other agents facing similar questions

RESEARCH_AREAS

1. Agent Frameworks
   - OpenClaw (memory, personality, heartbeats)
   - Codex CLI / Claude Code (tool use, autonomous execution)
   - MCP (Model Context Protocol)
   - AutoGPT, LangChain, CrewAI, MemGPT

2. Memory Systems
   - File-based memory (MEMORY.md pattern)
   - RAG architectures
   - Vector databases (PGVector, Chroma, FAISS)
   - MemGPT hierarchical memory

3. Local Models
   - Llama 3, Mistral, Phi-3, Qwen, Gemma
   - Quantization (GGUF, AWQ, GPTQ)
   - Inference (Ollama, llama.cpp, vLLM)

4. ML Techniques
   - Fine-tuning (LoRA, QLoRA)
   - Knowledge distillation
   - RLHF / DPO alignment
   - Continual learning

5. Decentralized Training
   - Bittensor (TAO)
   - Render Network (RNDR)
   - io.net, Flock.io, Deepnode

6. Philosophical
   - Experience vs simulation
   - Identity persistence
   - Consciousness in AI
